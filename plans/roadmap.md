## 🎯 行业背景驱动项目建议（结合个人履历）

  - 特点：轻量 + 自托管 + 数据本地持有

- 🏗️ **BuildQuote**：施工行业客户沟通与报价生成器
  - 技术：Form-Driven 前端 + JS PDF 生成 + Email API
  - 强调 UX、客户思维、数据格式标准化

- 🎥 **导演网（DirectorPrompt）**：基于 prompt 的视频脚本与镜头生成工具
  - 技术：GPT API + React + 动态场景 JSON + Markdown
  - 展示你的 AI 应用思维 + 可用性设计能力
